# ML Framework Selection — Recommendation System

## Overview
This project analyzes and justifies the selection of a Machine Learning framework
for building a product recommendation system for a retail company.

## My Activity Goal
- Analyze the requirements of a recommendation system
- Evaluate ML frameworks (TensorFlow, PyTorch, Scikit-learn)
- Justify the best framework choice based on the problem needs

## Project Structure
- `justification.md` → Full written justification and analysis
- `README.md` → Project overview

## Tools & Technologies
- Python
- ML Frameworks evaluated: TensorFlow, PyTorch, Scikit-learn
